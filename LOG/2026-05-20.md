#### 4. 🟡 보안: runner.py — 환경변수 전체 노출
- **파일:** `cli_anything/k_skill/runner.py`
- **문제:** `os.environ.copy()`로 모든 환경변수(API 키 등 시크릿 포함)를 하위 프로세스에 전달
- **수정:**
  - `_filter_env()` 헬퍼 함수 추가 — 허용 목록 기반 필터링
  - `_SUBPROCESS_ENV_ALLOWLIST`: PATH, HOME, LANG, NODE_PATH, PYTHONPATH, K_SKILL_ROOT 등 허용
  - `_SUBPROCESS_ENV_ALLOWLIST_PREFIXES`: LC_*, XDG_* 허용
  - `run_npm()`: `os.environ.copy()` → `_filter_env({"NODE_PATH": ...})`
  - `run_script()`: `os.environ.copy()` → `_filter_env({PYTHONPATH, K_SKILL_ROOT, **env_vars})`

#### 5. 🟡 보안: local_mcp_bridge.py — 환경변수 전체 노출
- **파일:** `cli_anything/k_skill/local_mcp_bridge.py:78`
- **문제:** `{**os.environ, **(self.env or {})}로 전체 환경변수를 하위 프로세스에 전달`
- **수정:** safe_keys 집합 기반 필터링 + LC_*/XDG_* 프리픽스 허용, 명시적 self.env만 추가 병합

#### 6. 🟡 보안: mcp_client.py — DNS rebinding 공격 가능
- **파일:** `cli_anything/k_skill/mcp_client.py:69-78`
- **문제:** 초기 URL 검증 시점에만 IP 체크 → DNS가 나중에 사설 IP로 해석되면 우회 가능
- **수정:**
  - `socket.getaddrinfo()`로 호스트명의 모든 IP 확인
  - 모든  resolved IP가 사설 범위인지 검사
  - DNS 해석 실패 시 MCPError 발생

#### 7. 🟢 품질: runner.py — K_SKILL_ROOT 하드코드 기본값
- **파일:** `cli_anything/k_skill/runner.py:25`
- **문제:** `~/nas_1tb/dev/k-skill`가 기본값으로 하드코드됨

#### 8. 🟢 품질: generate_proxy_skills.py — 미사용 스텝 정리
